## 사다리 게임 구현하기 1단계

PR 보낼 때 단계별 label(step-1 같이) 지정해서 보내기

### 사전지식

- 자바 기초 학습
- 타입, 조건문, 반복문, 2차원 배열
- 클래스, 객체, 메서드 이해하기

### 기능요구사항

- n명의 사람, m개의 사다리 개수 입력받기
- 사다리의 라인은 랜덤 값에 따라 있거나 없을 수도 있다?
    - 사다리가 있으면 -, 없으면 “ “(공백), 양 옆에 | 로 세로 표시
- 사다리 상태 화면에 출력하기. 어느 시점에 출력할 지는 자유
- 메서드 10줄 넘기지 말기
- 메서드 한 가지 일만 시키기
- 2차원 배열을 학습해라
